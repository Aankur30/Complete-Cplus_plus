import 'dart:io';

void main() {
  print('Enter your age:');
  int age = int.parse(stdin.readLineSync()!);

  if (age >= 13 && age <= 19) {
    print('You are a teenager');
  } else {
    print('You are not a teenager');
  }
}

Explaination:

The first line import 'dart:io'; imports the dart:io library, which provides functions for input and output.

The main function is where the program begins. The print function is used to prompt the user to enter their age.

The stdin.readLineSync() function reads the user's input from the console as a string. The int.parse() function is used to convert the string input to an integer.

The if statement checks if the age is between 13 and 19 (inclusive) using the && operator to combine two conditions. If the condition is true, the program prints "You are a teenager" using the print function. Otherwise, it prints "You are not a teenager".
